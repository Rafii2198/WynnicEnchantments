# Changes

- Bobby has been replaced with Distant Horizons, Bobby can still be installed as optional mod, bear in mind that both of these mods generate cache files which can take a lot of space
- Due to the change above, you may want to clear Bobby cache files so they don't take space on your computer if you don't plan on using both
- You will get prompt in game to download the latest version of the cache for Distant Horizons thanks to new WynnLODGrabber mod
- Predefined settings for Complementary Shaders + Euphoria Patches are now included
- More keybind changes
- Some Resourcepacks are now enabled by default
- Updated Wynntils to support 2.1.1
